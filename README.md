# CramAI

> Cram smarter, not harder

An AI-powered study tool that converts any video lecture into comprehensive notes, voice summaries, and interactive quizzes. Your ultimate cramming buddy.

---

## Features

- **Smart Text Notes** - AI extracts key concepts and creates structured summaries with headings and bullet points
- **Voice Summaries** - Convert notes to natural-sounding audio for on-the-go learning
- **Interactive Quizzes** - Auto-generated multiple-choice questions to test your knowledge
- **Fast Processing** - Process hour-long lectures in minutes
- **Export Options** - Download notes as PDF, voice summaries as MP3

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/sahilsaraswat/cramai.git

# Navigate to project directory
cd cramai

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be running at `http://localhost:5173`
